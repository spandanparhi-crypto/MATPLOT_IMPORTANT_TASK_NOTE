# ▶️ How to Run

Clone the repository

```bash
git clone https://github.com/your-username/Matplotlib-Important-Tasks.git
```

Move into the folder

```bash
cd Matplotlib-Important-Tasks
```

Install Matplotlib

```bash
pip install matplotlib
```

Run a file

```bash
python line_plot.py
```
